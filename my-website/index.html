Twitter Bot

I built a twitter bot that takes the top posts from the last 24 hours from reddit.com/r/earthporn and reddit.com/r/showerthoughts, runs a Processing sketch that generates an image out of those posts and uploads it to twitter.
It currently posts only when I run the program from my laptop, but I expect to setup a Raspberry Pi in the upcoming weeks and leave the bot running there, posting one picture every day.
